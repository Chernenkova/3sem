Сделано на основе прошлогоднего проекта по сбалансированным деревьям.
Программа строит дерево, балансирует его, а затем обходит двумя способами.
Ввод и вывод осуществляются через файлы.
Если потребуется не проводить балансировку при построении, то в файле tree.c в функции tree_add (самая первая) необходимо
изменить возвращаемое дерево на tree, т.е. return tree (тогда при добавлении элемента функция балансировки вызываться не
будет).
